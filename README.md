Array with Animation Offset for blender by Julbogje

Installation: open Blender --> Edit --> Preferences --> Add-ons --> Install from Disk --> select array_with_animation_offset
To use it: open sidebar --> press Array Anim Offset --> enter name / drag & drop the object you want to create an array for into Base Object --> press Create Animated Array

The number of Frame Offset gives each instance an offset of the number you entered, the multiply (x) influences the offset for ease in and/or out
The rest of the settings work about the same as the Array modifier in Blender

This Add-on creates instances of an object, unless the object has shape keys in which it will create linked copies
